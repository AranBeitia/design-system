Intro

This pattern library represents a standard set of web components / patterns that has been designed and built through user testing, accessibility testing and meeting the AXA GWAPA frontend coding guidelines. It is not exhaustive and should not be viewed as being exhaustive; it's a living, breathing, pull-request loving, store of how we, as an organisation aim to build and craft our web experiences.

If you find a bug or have an improvement, please do fork this project on github.

Thank you
